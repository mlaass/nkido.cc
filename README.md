# NKIDO.cc

The public project website for [NKIDO](https://github.com/mlaass/nkido) — high-performance live-coded audio synthesis.

## Development

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build
```

## Tech Stack

- SvelteKit with adapter-static
- Svelte 5 runes
- Bun package manager
- Netlify deployment
